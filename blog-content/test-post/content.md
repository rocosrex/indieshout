# IndieShout 테스트 포스트

IndieShout 블로그 자동 게시 테스트입니다.

## 주요 기능

- ✅ X (Twitter) 연동
- ✅ Threads 연동
- ✅ Hugo 블로그 자동 게시
- ✅ 한글 → 영문 자동 번역
- ✅ AWS S3 이미지 업로드

## 사용 방법

```bash
indieshout blog publish-folder test-post
```

간단하게 블로그와 SNS에 동시 게시할 수 있습니다!
